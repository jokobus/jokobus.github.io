Introduction

This article gives a quick overview over the feedforward phase of a Neural Network.

Nodes

A Neural Network is made of many Nodes that are connected to each other. Each Node itself is a very basic unit that does only do three things.

  1. Multiply each input with a weight and sum all inputs up. $y_{in} = \sum_{i=0}^m w_i\cdot x_i$, where $m$ is the number of inputs.
  2. Process the single, scalar input value by some activation function $y_{out}=f(y_{in})$ where $y_{in}$ is the result from step 1 and $y_{out}$ is the result from step 1 after being applied to the activation function.
  3. Output the result $y_{out}$ to the next node

So, a single node pretty much looks like this:

Figure 1: A single Node with its weights

Here:

  1. $y_{in} = w_1x_1 + w_2x_2 + w_0x_0$ (where each x_i and w_i is a scalar of course)
  2. $y_{out} = f(y_{in})$

Bias

One special input we have to talk about is the bias $x_0$. Per definition $x_0 = 1$ always. The bias is needed as an offset. You can think of it as the multi-dimensional equivalent to the y-intercept in 2D. Without a y-intercept each function in 2D has to go through the center of the coordinate system. The same applies for our Node: With this offset, the Neural Network can e.g. classify data even if their optimal separation hyperplane does not go through the origin.

Example

Let’s say we have the following Node and the activation function $f(x) = \begin{cases} 1 & \text{if } x > 0\\ 0 & \text{else } \end{cases}$

Figure 2: Example of single Node with its weights

  1. $y_{in} = \sum_{i=0}^m w_i\cdot x_i = w_1x_1 + w_2x_2 + w_0x_0 = 1\cdot 2 + 2\cdot 4 + 3 \cdot 1 = 2+8+3 = 13$
  2. $y_{out} = f(y_{in})$. We have $y_{in} = 13 > 0$ so $y_{out} = 1$
  3. The node outputs $y_{out} = 1$

Activation functions

In the previous example we used the so called binary step function as our activation function. However, there are a few more I would like to mention here.

FunctionFormulaDescription
Identify function$f(x)=x$Very basic
Binary step function$f(x) = \begin{cases}1 & \text{if } x > 0\\0 & \text{else }\end{cases}$
Bipolar step function$f(x) = \begin{cases}1 & \text{if } x > 0\\-1 & \text{else }\end{cases}$
Binary Sigmoid$f(x) = \frac{1}{1+e^{-x}} = \frac{e^x}{e^x+e^{-x}}$
Bipolar Sigmiod$f(x) = \frac{2}{1+e^{-x}}+1 = \frac{e^x-e^{-x}}{e^x+e^{-x}} = tanh(x)$
ReLu (Rectified Linear Unit )$f(x) = max(x,0)$
Softplus$f(x) = ln(1+e^x)$
Leaky ReLu$f(x)=\begin{cases}x & \text{if } x > 0\\\alpha x, \;\alpha\text{ small} & \text{else }\end{cases}$
Softmax$f(x_i)=\frac{e^{x_i}}{\sum_j e^{x_j}}$

Architecture

Now we have seen how each node works. He gets some information, processes this information and outputs the result. That is not very impressing so far, but many of those nodes are combined in a specific Architecture, then they can perform powerful tasks.

Feedforward

The following plot shows a very basic Neural Network, that has 3 layers. One input layer $x$, one hidden layer $z$ and one output layer $y$. Between each consecutive layer we have connections $w$ and $v$. For the moment, we don’t care about how the weights were computed, as this is done by the training process. The feedforward part is used for predicting, the output for a given input. If we are testing the Neural Network, we use the feedforward method, as shown in this article. In the training part, we also use feedforward, but that is a topic for another article: Backpropgation

Figure 4: Neural Network with 3 layers

Each connection $w_{ij}$ is the weight from node $x_i$ to $z_j$. So, for example $w_{01}$ is the weight from $x_0$ (the bias!) to $z_1$.

Steps of Feedforward

  • For each layer (starting at the first hidden layer)
    • For each node in the layer
      1. $z_{in} = \sum_{i=0}^m w_i\cdot x_i$, where $m$ is the number of inputs. Resp. $y_{in} = \sum_{i=0}^p v_i\cdot z_i$, where $p$ is the number of inputs.
      2. Apply the activation function $z_{out}=f(z_{in})$, resp. $y_{out}=f(y_{in})$
      3. Output the result $z_{out}$ to the next layer, resp. $y_{out}$ to the output
    • End
  • End

Example

Let’s consider the following Neural Network. Each node has the binary step function $f(x) = \begin{cases}1 & \text{if } x > 0\\0 & \text{else }\end{cases}$ applied to it.

Figure 5: Example of a Neural Network with 3 layers

  • We begin with the hidden layer
    1. First we calculate all $z_{in}$ (Note that $x_0$ and $z_0$ are the biases, so $x_0=1$ and $z_0=1$ always. )
      • $z_{1,in} = w_{01}x_0+w_{11}x_1+w_{21}x_2 = 1\cdot 1+2\cdot 2+(-0.5)\cdot 3 = 3.5$
      • $z_{2,in} = w_{02}x_0+w_{12}x_1+w_{22}x_2 = 0\cdot 1+3\cdot 2+(-2)\cdot 3 = 0$
      • $z_{3,in} = w_{03}x_0+w_{13}x_1+w_{23}x_2 = 4\cdot 1+(-4)\cdot 2+1\cdot 3 = -1$
    2. Apply the activation function $f(x) = \begin{cases}1 & \text{if } x > 0\\0 & \text{else }\end{cases}$
      • $z_{1,out} = f(z_{1,in}) = f(3.5)= 1$
      • $z_{2,out} = f(z_{2,in}) = f(0) = 0$
      • $z_{3,out} = f(z_{3,in}) = f(-1)= 0$
  • We repeat the same for the output layer
    1. Calculate all $y_{in}$
      • $y_{1,in} = v_{01}\cdot z_0+v_{11}\cdot z_{1,out}+v_{21}\cdot z_{2,out}+v_{31}\cdot z_{3,out} =0.25\cdot 1+2\cdot 1+1.5\cdot 0+0.75\cdot 0 = 2.25$
      • $y_{2,in} = v_{02}\cdot z_0+v_{12}\cdot z_{1,out}+v_{22}\cdot z_{2,out}+v_{32}\cdot z_{3,out} =(-1)\cdot 1 + 0.5\cdot 1+1\cdot 0 + 1.25\cdot 0 = -0.5$
    2. Apply the activation function
      • $y_{1,out} = f(y_{1,in}) = f(2.25)= 1$
      • $y_{2,out} = f(y_{2,in}) = f(-0.5) = 0$

Tadaa! This is our final output. Our net predicts $y_1 = 1$ and $y_2 = 0$

Feedforward in Matrix Calculus

As you already may have guessed the whole process we did before can be simplified by using Matrix Calculus.

Figure 6: Simplified Neural Network to demonstrate the Matrix Calculus

The steps are pretty much the same:

  • For each layer (starting at the first hidden layer)
    1. $z_{in} = W^T x$
    2. Apply the activation function $z_{out}=f(z_{in})$
    3. Output the result $z_{out}$
  • End

So, the little bit more detailed steps for our Neural Network in Figure 6 are:

  1. $z_{in} = W^Tx$
  2. $z_{out} = f(z_{in})$
  3. $y_{in} = V^Tz$
  4. $y_{out} = f(y_{in})$ As you can see this looks way more handsome than taking the sum over all input value multipiled with their weights.

Dimensions

Hereby we have to consider the dimensions of the different layers. If $x\in \R^{m\times 1}$, $z\in \R^{p\times 1}$ and $y\in \R^{n\times 1}$ then $W\in \R^{m\times p-1}$ and $V\in \R^{p\times n}$.

Example

Let’s do the example from Figure 5 another time. We use the identity activation function again, but do the math in matrix form this time.

First of all, let’s determine $W$ and $V$.

$W=\begin{bmatrix} w_{01} & w_{02} & w_{03} \\ w_{11} & w_{12} & w_{13} \\ w_{21} & w_{22} & w_{23} \end{bmatrix} \text{, } V=\begin{bmatrix} v_{01} & v_{02} \\ v_{11} & v_{12} \\ v_{21} & v_{22} \\ v_{31} & v_{32} \end{bmatrix}$

Just for clarity, we check the dimensions: $x\in \R^{3\times 1}$, $z\in \R^{4\times 1}$ and $y\in \R^{2\times 1}$ then $W\in \R^{3\times 3}$ and $V\in \R^{4\times 2}$.

If we fill in the values, then we get $W=\begin{bmatrix} 1 & 0 & 4 \\ 2 & 3 & -4 \\ -0.5 & -2 & 1 \end{bmatrix} \text{, } V=\begin{bmatrix} 0.25 & -1 \\ 2 & 0.5 \\ 1.5 & 1 \\ 0.75 & 1.25 \end{bmatrix}$

Let’s do the steps:

  1. $z_{in} = W^T x = \begin{bmatrix} 1 & 0 & 4 \\ 2 & 3 & -4 \\ -0.5 & -2 & 1 \end{bmatrix} \begin{bmatrix} 1\\ 2\\ 3 \end{bmatrix} = \begin{bmatrix} 3.5 \\ 0 \\ -1 \end{bmatrix}$

  2. $z_{out}=f(z_{in})= \begin{bmatrix} f(z_{1,in})\\ f(z_{2,in})\\ f(z_{3,in}) \end{bmatrix} = \begin{bmatrix} f(3.5)\\ f(0)\\ f(-1) \end{bmatrix} = \begin{bmatrix} 1\\ 0\\ 0\\ \end{bmatrix} $

  3. $y_{in} = V^T z_{out} = \begin{bmatrix} 0.25 & 2 & 1.5 & 0.75 \\ -1 & 0.5 & 1 & 1.25 \end{bmatrix} \begin{bmatrix} 1\\1\\0\\0 \end{bmatrix} = \begin{bmatrix} 2.25\\-0.5 \end{bmatrix} $

    Note that in step 3 we used the same $z_{out}$ as in step 2, but added the bias $z_0$ at the “top”.

  4. $y_{out} = f(y_{in}) = \begin{bmatrix} f(y_{1,in})\\ f(y_{2,in})\\ \end{bmatrix} = \begin{bmatrix} f(2.25)\\ f(-0.5)\\ \end{bmatrix} = \begin{bmatrix} 1\\ 0\\ \end{bmatrix} $

This is the same result as if we had used the normal way, without matrix calculations.
