Custom Anki Card

Howto create a custom Anki Card with three fields and cross-learning.

Today, I came accross my old Notes from my German Class at School. I decided to learn the stylistic devices (dt. “Stilmittel”) again. To do so, I want to use Anki and create a custom card with the three fields Name, Definition and Example. In this guide, I will explain how to create a custom Anki card (template) with multiple fields and cross-learning. When the cards are created, I have to learn each three times.

  1. Name is given, find Definition and Example
  2. Definition is given, find Name and Example
  3. Example is given, find Name and Definition

1. Create Template

  1. Go to Tools>Manage Note Types (Ctrl+Shift+N)
  2. Click Add
  3. Choose Add:Basic (and reversed card)
  4. Name it something like Cross Learning (Name, Definition, Example)
  5. Select the newly created Note Type in the list (scroll down)
  6. Click Fields
  7. Rename and Add, so that you have 1:Name, 2:Definition, 3:Example. Save
  8. Click Cards

    1. Choose Card Type 1

      • Front Template

        {{Name}} is a placeholder for the content written in the Name field.

        Wrapping it by {{#Name}} … {{/Name}} ensures that the card with the Name of the stylistic device on the front is only created if Name is non-empty.

        {{#Name}}
   {{Name}}
{{/Name}}

      • Back Template

        {{Name}}<br><br>
{{Definition}}<br><br>
{{Example}}

    2. Choose Card Type 2

      • Front Template
        {{#Definition}}
   <br><br>
   {{Definition}}
{{/Definition}}
      • Back Template
        {{Name}}<br><br>
{{Definition}}<br><br>
{{Example}}

    3. Click on Options (upper right corner) and Add Card Type

    4. Choose Card Type 3

      • Front Template
        {{#Example}}
   <br><br><br><br>
   {{Example}}
{{/Example}}
      • Back Template
        {{Name}}<br><br>
{{Definition}}<br><br>
{{Example}}
  9. Save all

2. Create a new Card using the Template

  1. Choose Cross Learning (Name, Definition, Example) as your Card Type

  2. Add context

  3. Save

  4. Repeat for more Cards

3. Learn

When you are done with creating cards you can start to learn them. You will be asked every card three times, with only the Name or the Definition or the Example given. (This sentence is a Polysyndeton by the way). You can also leave fields empty, then Anki won’t ask you those fields.

Have fun learning stylistic devices or any other content with the same structure.
