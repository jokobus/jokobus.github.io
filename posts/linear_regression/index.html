Overview

Why and when do we need linear regression? In a nutshell, linear regression is used to predict data that has a linear relationship. That means all the datapoints are roughly on a straight line (in 2D), resp. on a hyperplane (in 3D). Firstly, we find some expression for the linear relationship (e.g. in 2D: $y=a\cdot x + b$; where $a$ is the slope and $b$ is the y-intercept). Secondly, we use this expression to insert some data where we only know the $x$ value, but want to predict their unknown $y$ value.

Linear regression with one variable

To get a feeling and understanding what linear regression is, we start with the most simple version: The 2D linear regression with one variable.

Figure 1: Some example data with a linear relationship and a regression line.

Figure 1 shows some datapoints, which are roughly linearly lined up. The expression for the blue regression line is: $y=a\cdot x + b$; where $a$ is the slope and $b$ is the y-intercept.

Find best regression line

First, we find $a$ and $b$ such that some error is minimized. Most datapoints are not on exactly on the regression line. Thus the $x$-$y$-relationship of each datapoint $i$ is not simply $y_i=a\cdot x_i + b$, but actually $y_i=a\cdot x_i + b + \varepsilon _i$ where $\varepsilon _i$ is the error we want to minimize. We can rewrite the expression and get $\varepsilon_i = y_i - (ax_i+b)$ where $y_i$ is the actual value and $- (ax_i+b)$ is the predicted value.

Now, what do we actually want to achieve? We want to minimize the total error $\varepsilon$ which is some ‘compound’ of all $\varepsilon_i$.

To get $\varepsilon$ we could try one of the following:

  • add up all $\varepsilon_i$ -> not a good idea as positive and negative $\varepsilon_i$ cancel each other
  • add up the absolute value of all $\varepsilon_i$ -> not that smart, because $\varepsilon$ won’t be differentiable anymore
  • add up the squares: $\varepsilon_i^2$ -> That is the so called least squared method (aka Euclidean Norm). Perfect!

Next, we compute the least squares. We want to minimize some function $f(a,b)$.

$f(a,b) = \sum_{i=1}^n \varepsilon_i^2 = \sum_{i=1}^n (y_i-ax_i-b)^2$, where $n$ is the number of $x$-$y$-pairs we have

We take the derivatives wrt. to $a$ and $b$ and set them equal to zero.

I) $f_a = \frac{\partial f(a,b)}{\partial a} = \sum_{i=1}^n -2 (y_i-ax_i-b) x_i = 0$

II) $f_b = \frac{\partial f(a,b)}{\partial b} = \sum_{i=1}^n -2 (y_i-ax_i-b) = 0$

Then we do the math: (Note that we simply ommit the -2 first)

I) $0 =\sum y_i x_i - \sum a x_i^2 - \sum b x_i = \sum y_i x_i - a \sum x_i^2 - b\sum x_i $

II) $0 =\sum y_i - \sum a x_i - \sum b = \sum y_i - a \sum x_i - nb $

from II) $nb = \sum y_i - a \sum x_i$

II) in I) $0 =\sum y_i x_i - \sum a x_i^2 - (\frac{\sum y_i}{n} - \frac{a\sum x_i}{n})$

$\sum y_i x_i - \frac{\sum x_i\cdot \sum y_i}{n} = a (\sum x_i^2 - \frac{\sum x_i \cdot \sum x_i}{n})$

$ a = \frac{\sum x_i \sum y_i - n \sum x_iy_i}{(\sum x_i)^2 - n \sum x_i^2}$

$ b = \frac{\sum y_i - a \sum x_i}{n}$

That’s it. Now we have two expressions how we can find $a$ and $b$ of a given dataset.

Everything we need to do is to plug in all the data into the formulas for $a$ and $b$. Then we can take some unknow datapoint where we only have $x_i$, but not the corresponding $y_i$. We insert $x_i$ into our formula of $y_i = ax_i+b$ and we will get the $y_i$ value.

In case we leave 2D and want to solve problems in higher dimensions, we could do exactly the same procedure. In 3D we have $y_i = ax + bz + c$ and can solve the three resulting equalities to find some formulas for $a$, $b$ and $c$. However, the more dimensions we get, the more computation expensive this will become. Luckily, there is a smarter way, the Normal Equation of Linear Regression. We will come to that later.

Example

Consider the following five datapoints: (0,1), (1,1), (2,2), (3,2), (4,3)

$ a = \frac{\sum x_i \sum y_i - n \sum x_iy_i}{(\sum x_i)^2 - n \sum x_i^2} = \frac{5\cdot(0\cdot 1 + 1\cdot 1 + 2\cdot 2 + 3\cdot 2 + 4\cdot 3)- (0+1+2+3+4)\cdot (1+1+2+2+3)}{5\cdot(0^2+1^2+2^2+3^2+4^2)-(0+1+2+3+4)^2} = \frac{115-90}{150-100} = \frac{25}{50} = 0.5$

$ b = \frac{\sum y_i - a \sum x_i}{n} = \frac{1+1+2+2+3-0.5\cdot(0+1+2+3+4)}{5} = \frac{9-5}{5} = 0.8$

Thus, the final equation is $y = 0.5x+0.8$

If we want to predict the y-value for $x=2$, we get: $y=0.5\cdot2+0.8 = 1.8 \not = 2$. That is correct and makes sense, as our regression line does not perfectly go through each point, but rather minimizes the overall error.

Figure 2: Our example’s datapoints with their regression line.

Assumptions

The previous example worked very well. However, our data needs to fulfill specific requirements.

  1. Linearity: A linear relationship between all $x$ and $y$ datapoints
  2. Homoscedasticity: The variance of error is independent of $x$
  3. Normality: The data is normally distributed
  4. Independence: Each measurements/datapoint is independent from other measurements/datapoints

If those four requirements are fulfilled, we can use linear regression.

Polynomial Regression of order m=2 (quadratic)

We have $y = a_2x^2+a_1x^1+a_0 + \varepsilon$ and the next steps are actually pretty close to what we did above to find the best regression line.

The sum of the squares of the errors $S = \sum_i \big((y_i-a-bx_i-cx_i^2)^2\big)$. Differentiating wrt. $a$, $b$ and $c$ we get:

I) $an + bS_1+cS_2 = S_5$

II) $aS_1 + bS_2+cS_3 = S_6$

III) $aS_2 + bS_3+cS_4 = S_7$

where $S_1 = \sum_i x_i$

$S_2 = \sum_i x_i^2$

$S_3 = \sum_i x_i^3$

$S_4 = \sum_i x_i^4$

$S_5 = \sum_i y_i$

$S_6 = \sum_i x_iy_i$

$S_7 = \sum_i x_i^2y_i$

Do you see the pattern?

Polynomial regression for any order m

If yes, then the following might be interesting to you. It is (with a small modification) taken from the Wikipedia-Article about Polynomial Regression . Solving this linear system for the $a_i$’s will work for any polynomial power $m$.

$ \left[ \begin{matrix} \sum_{i=1}^n x_i^0 & \sum_{i=1}^n x_i^1 & \sum_{i=1}^n x_i^2 & \cdots & \sum_{i=1}^n x_i^m \\ \sum_{i=1}^n x_i^1 & \sum_{i=1}^n x_i^2 & \sum_{i=1}^n x_i^3 & \cdots & \sum_{i=1}^n x_i^{m+1} \\ \sum_{i=1}^n x_i^2 & \sum_{i=1}^n x_i^3 & \sum_{i=1}^n x_i^4 & \cdots & \sum_{i=1}^n x_i^{m+2} \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ \sum_{i=1}^n x_i^m & \sum_{i=1}^n x_i^{m+1} & \sum_{i=1}^n x_i^{m+2} & \cdots & \sum_{i=1}^n x_i^{2m} \\ \end{matrix} \right] $ $ \left[ \begin{matrix} a_0 \\ a_1 \\ a_2 \\ \vdots \\ a_m \\ \end{matrix} \right] $ $=\left[ \begin{matrix} \sum_{i=0}^n y_i x_i^0 \\ \sum_{i=0}^n y_i x_i^1 \\ \sum_{i=0}^n y_i x_i^2 \\ \vdots \\ \sum_{i=0}^n y_i x_i^m \\ \end{matrix} \right] $

Normal Equation of Linear Regression

So far we mainly talked about linear regression in 2D and realized that doing linear regression in higher dimensions is analog to 2D, but very includes a lot of work. Luckily, as already mentioned, there is an easier way to do linear regression for higher dimensions.

Normal Equation of Linear Regression in 2D

Derivation

$\vec u=\vec b-x\vec a$

min $\frac12 (\vec b - x\vec a)^2$

$\frac{d}{d\vec a} \frac12 (\vec b - x\vec a)^2 = (\vec b - x\vec a)\vec a = 0$

$ \vec b\cdot\vec a - x\vec a\cdot\vec a = 0$

$x = \frac{\vec b\cdot\vec a}{\vec a\cdot\vec a}$

To get the general solution for the least squares method in n-dimensions we do the following steps:

$\varepsilon(x) = (\vec a x - \vec b)^2 = (\vec a x - \vec b)^T (\vec a x - \vec b) = (\vec a x)^T (\vec a x) - (\vec a x)^T \vec b - \vec a x \vec b^T + \vec b^T\vec b = x^2\vec a^T\vec a - x(\vec a^T\vec b+\vec b^T\vec a)+\vec b^T\vec b = x^2\vec a^T\vec a - 2x\vec a^T\vec b+\vec b^T\vec b$

$\frac{\partial \varepsilon}{\partial x} = 2x\vec a^T\vec a -2\vec a^T\vec b = 0$

$x\vec a^T\vec a -\vec a^T\vec b = 0$

Th gives the general solution for the least squares method in n-dimensions:

$$x=\frac{\vec a^T\vec b}{\vec a^T\vec a}$$

Example

Let $\vec a = \begin{pmatrix} 2 \\ 3 \\ 4 \\\end{pmatrix}$ and $\vec b = \begin{pmatrix} 1 \\ 2 \\ 3 \\\end{pmatrix}$ then $\varepsilon (x) = (2x-1)^2+(3x-2)^2+(4x-3)^2 \rightarrow \frac{\partial \varepsilon}{\partial x} = 2(2x-1)+2(3x-2)+2(4x-3) = 0 \Leftrightarrow (2x-1)+(3x-2)+(4x-3) = 0$. This equation can be solved for $x$ and you will get $x=\frac23$.
